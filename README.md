# Nonlinear Interactions in the Characteristics of Real Assets

MSc Finance supervised project, HEC Montréal (November 2025)
Director: Prof. Anthony Sanford

## Summary
Tested whether nonlinear dynamics and macroeconomic state-dependence
drive real asset returns, extending equity asset pricing literature
to REITs and commodities.

Built panel OLS framework in Python on 25+ years of U.S. REIT and
commodity data (Bloomberg, FRED), with macroeconomic state variables
constructed via PCA on 20+ macro series.

## Key findings
- Non-monotonic "valuation corridor" in REIT FFO yields
- Leverage penalty intensifies during periods of macroeconomic uncertainty
- Nonlinear specifications nearly doubled explanatory power for REITs
  (within-R² to 20.6%)
